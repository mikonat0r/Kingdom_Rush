# Проект: Kingdom Rush Tower Defense

## 1. Тема проекта
Консольный симулятор управления защитными башнями по мотивам культовой игры **Kingdom Rush**. Программа позволяет возводить оборонительные сооружения, прокачивать их характеристики по каноничным ценам, отдавать тактические приказы и продавать башни для возврата золота в казну.

## 2. Архитектура и ООП составляющие

### Абстрактный класс:
* `Tower` — базовый класс для всех защитных сооружений. Содержит общие поля (`id`, `name`, `cost`, `level`) и чисто виртуальные методы `showInfo()` и `getType()`.

### Дочерние классы (Реализации башен):
1. `ArcherTower` — башня лучников (высокая скорость атаки, физический урон).
2. `MageTower` — магическая башня (высокий урон, пробивающий тяжелую броню).
3. `BarracksTower` — казармы (нанимает воинов Militia/Footmen/Knights для удержания врагов).
4. `ArtilleryTower` — артиллерия гномов (наносит огромный AOE-урон по площади).

### Интерфейсы (Чисто виртуальные классы):
1. `IAttackable` — реализует метод `attack()` для боевых башен.
2. `IUpgradable` — реализует метод `upgrade()` для повышения тира/уровня постройки.
3. `ICreatable` — реализует метод `createUnit()` специально для казарм (призыв гарнизона).

### Управляющий класс:
* `KingdomManager` — ядро системы. Хранит внутри себя динамический кошелек с золотом (`gold`), вектор умных указателей `std::vector<std::shared_ptr<Tower>>`, отвечает за интерактивное меню, экономический баланс покупок/апгрейдов и каноничную систему сноса (продажи) башен за 70% стоимости.

## 3. Использование Полиморфизма
Полиморфизм реализован при хранении разнородных объектов (всех 4-х типов башен) в едином векторе `std::vector<std::shared_ptr<Tower>> mapTowers`. 
* При выводе списка башен динамически вызывается переопределенный метод `getType()`.
* Приказы способностей и улучшения распределяются полиморфно через безопасное приведение типов `dynamic_cast` прямо во время работы программы.

## 4. Как запустить проект
1. Откройте файл решения `.sln` в Visual Studio.
2. Убедитесь, что конфигурация сборки установлена на `x64` или `x86`.
3. Нажмите `F5` для запуска консольного приложения.
